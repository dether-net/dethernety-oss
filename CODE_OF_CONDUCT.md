# Code of Conduct

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

All participants in the Dethernety community -- contributors, maintainers, and users interacting through issues, pull requests, discussions, or other project spaces -- are expected to read and follow the full text at the link above.

## Reporting

If you experience or witness unacceptable behavior, report it through [GitHub Security Advisories](https://github.com/dether-net/dethernety-oss/security/advisories/new). Reports are handled confidentially.

## Enforcement

Project maintainers will review all reports and respond as they see fit, following the enforcement guidelines in the Contributor Covenant. Maintainers who do not follow the Code of Conduct may face consequences determined by other members of the project leadership.
